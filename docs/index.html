🎮 התקנה אוטומטית — Elitzur Games

⬇ הורד את ה-Installer

מזהה את מערכת ההפעלה...

איך להשתמש:
  1. וודא ש-Minecraft הופעל פעם אחת לפחות (כדי שה-launcher יוריד Java)
  2. לחץ על הכפתור הירוק למעלה — יורד קובץ קטן
  3. לחץ עליו פעמיים — חלון יקפוץ ויתקין הכל אוטומטית
  4. אחרי שמופיע "ההתקנה הסתיימה" — פתח Minecraft Launcher, בחר את הפרופיל "Elitzur Games (Fabric 26.1.2)" ולחץ Play
⚠ macOS: אם Mac חוסם פתיחה של הקובץ — לחץ קליק ימני → Open ואשר.
⚠ Windows: אם Windows מציג "Windows protected your PC" — לחץ "More info" → "Run anyway".

מה ההתקנה כוללת

  • Simple Voice Chat — דיבור עם שחקנים
  • Xaero's Minimap — מפת מיני
  • Sodium + Sodium Extra — יותר FPS
  • LambDynamicLights — אור דינמי
  • Emotecraft — אנימציות תנועה
  • Zoomify — זום עם C
  • Fullbright UB — תמיד אור
  • Golden-Carrot Hunger Bar
  • Fancy Heart Bar
  • Fabric Loader + Fabric API
התקנה ידנית (אם האוטומטית לא עובדת)
שלב 1: התקן Fabric Loader
https://fabricmc.net/use/installer/
הורד, פתח, בחר Minecraft Version: 26.1.2, לחץ Install.
שלב 2: פתח את תיקיית mods
Windows: %APPDATA%\.minecraft\mods\
macOS: ~/Library/Application Support/minecraft/mods/
שלב 3: הורד לתיקיית mods:
  • Fabric API
  • Simple Voice Chat
  • Xaero's Minimap
  • Sodium
  • Sodium Extra
  • LambDynamicLights
  • Emotecraft
  • Zoomify
  • Player Animation Library (תלות)
  • Fabric Language Kotlin (תלות)
  • YACL (תלות)
שלב 3ב: הורד לתיקיית resourcepacks:
  • Fullbright UB
  • Golden-Carrot Hunger Bar
  • Fancy Heart Bar
אחר כך הפעל את שלושתם ב-Options → Resource Packs.
שלב 4: ב-launcher — בחר פרופיל "fabric-loader-26.1.2" ולחץ Play.
התקנה דרך פקודה בטרמינל

Windows (PowerShell)

Set-ExecutionPolicy -Scope Process Bypass -Force; irm https://elitzurms-art.github.io/elitzur-mods-install/install.ps1 | iex

macOS (Terminal)

curl -fsSL https://elitzurms-art.github.io/elitzur-mods-install/install.sh | bash
