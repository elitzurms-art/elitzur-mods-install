  • Simple Voice Chat — דיבור עם שחקנים אחרים
  • Xaero's Minimap — מפת מיני בפינת המסך
  • Fullbright UB — תמיד יש אור, גם במערות
  • Golden-Carrot Hunger Bar — שדרוג ויזואלי לבר רעב
  • Fancy Heart Bar — שדרוג ויזואלי לבר חיים
  • Fabric Loader + Fabric API (התשתית של פבריק)

בחר את מערכת ההפעלה שלך:

התקנה אוטומטית — Windows

⚠ חשוב: הפעל את Minecraft פעם אחת לפני שאתה ממשיך, כדי שה-launcher יוריד Java.
  1. לחץ Win+X ובחר "Windows PowerShell" (לא Terminal, לא CMD)
  2. העתק והדבק את הפקודה הזו, ולחץ Enter:
Set-ExecutionPolicy -Scope Process Bypass -Force; irm https://elitzurms-art.github.io/elitzur-mods-install/install.ps1 | iex
  1. חכה עד שיופיע "ההתקנה הסתיימה"
  2. פתח את Minecraft Launcher
  3. בחר את הפרופיל "Elitzur Games (Fabric 26.1.2)"
  4. לחץ Play והתחבר לשרת — תראה אייקון מיקרופון בפינה

התקנה אוטומטית — macOS

⚠ חשוב: הפעל את Minecraft פעם אחת לפני שאתה ממשיך.
  1. פתח את Terminal (אפליקציה ב-Applications → Utilities)
  2. העתק והדבק את הפקודה הזו, ולחץ Enter:
curl -fsSL https://elitzurms-art.github.io/elitzur-mods-install/install.sh | bash
  1. חכה עד שיופיע "ההתקנה הסתיימה"
  2. פתח את Minecraft Launcher
  3. בחר את הפרופיל "Elitzur Games (Fabric 26.1.2)"
  4. לחץ Play והתחבר לשרת

התקנה ידנית (אם האוטומטי לא עובד)

שלב 1: התקן Fabric Loader
https://fabricmc.net/use/installer/
הורד, פתח, בחר Minecraft Version: 26.1.2, לחץ Install.
שלב 2: פתח את תיקיית mods
Windows: %APPDATA%\.minecraft\mods\ (אם לא קיימת — צור)
macOS: ~/Library/Application Support/minecraft/mods/
שלב 3: הורד לתיקיית mods:
  • Fabric API (0.150.0+26.1.2)
  • Simple Voice Chat (2.6.18+26.1.2)
  • Xaero's Minimap (fabric 26.1.2-25.3.14)
שלב 3ב: הורד לתיקיית resourcepacks:
  • Fullbright UB
  • Golden-Carrot Hunger Bar
  • Fancy Heart Bar
אחר כך הפעל את שלושתם ב-Options → Resource Packs.
שלב 4: ב-launcher — בחר פרופיל "fabric-loader-26.1.2" ולחץ Play.
